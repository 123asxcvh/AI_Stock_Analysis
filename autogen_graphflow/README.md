# 股票AI分析系统 - 基于AutoGen GraphFlow

这是一个专门为股票投资分析设计的智能Agent系统，基于AutoGen GraphFlow框架构建，提供全面的股票分析能力。

## 系统架构

### 智能体团队结构

```
📋 计划规划师 (planner_agent)
    ↓
┌─────────────────────────────────────┐
│ 📈 技术分析师  │ 💰 基本面分析师 │
│ technical_analyst │ fundamental_analyst │
├─────────────────────────────────────┤
│ 📰 舆情分析师  │ ⚠️ 风险评估师  │
│ sentiment_analyst │ risk_analyst   │
└─────────────────────────────────────┘
    ↓
📝 报告生成器 (report_generator)
```

### 核心功能

1. **智能任务规划** - 计划规划师制定详细的分析策略
2. **并行多维分析** - 四个专业分析师同时工作，提高效率
3. **深度网络搜索** - 集成Exa搜索、Chrome浏览器等工具
4. **专业报告生成** - 生成结构化的投资分析报告

## 使用方法

### 基本用法

```bash
# 分析默认股票（600519 贵州茅台）
python main.py

# 分析指定股票
python main.py 000001

# 分析其他股票
python main.py 000002
```

### 高级用法

```bash
# 使用原始模板
python main.py

# 运行测试
python test_simple.py
```

## 配置说明

### API配置
在 `config.py` 中配置：
- GLM-4.6 API密钥
- 模型参数（温度、超时时间等）
- MCP服务器配置

### MCP工具集成

系统集成了多个MCP工具：
- **exa**: 网络搜索
- **chrome**: 浏览器自动化
- **fetch**: 数据获取
- **sequential-thinking**: 深度思考

## 分析维度

### 1. 技术面分析
- 价格走势和技术形态
- 技术指标分析（MACD、KDJ、RSI、布林带等）
- 成交量和资金流向分析
- 支撑位和阻力位判断
- 基于"少妇战法"的B1、B2买点分析

### 2. 基本面分析
- 财务报表分析
- 关键财务指标（PE、PB、ROE等）
- 行业地位和竞争优势
- 管理层评估
- 估值模型分析

### 3. 舆情分析
- 新闻舆情监控
- 社交媒体情绪分析
- 研究报告汇总
- 市场情绪指标
- 重大事件影响评估

### 4. 风险评估
- 系统性风险分析
- 非系统性风险评估
- 流动性风险
- 波动率风险
- 下行风险保护策略

## 输出报告格式

生成的报告包含：
1. **执行摘要** - 核心结论和投资建议
2. **技术面分析** - 技术信号和买卖点
3. **基本面分析** - 估值分析和投资价值
4. **舆情分析** - 市场情绪和催化剂
5. **风险评估** - 风险点和控制措施
6. **投资建议** - 具体策略和操作建议
7. **免责声明** - 风险提示

## 注意事项

1. **API密钥配置** - 确保GLM-4.6 API密钥已正确配置
2. **网络连接** - 需要稳定的网络连接进行数据搜索
3. **数据源** - 部分分析依赖外部数据源，可能需要额外配置
4. **免责声明** - 本系统仅供学习研究，不构成投资建议

## 扩展开发

### 添加新的分析师Agent

1. 在 `config.py` 中添加新的智能体配置
2. 在 `prompt.py` 中定义提示词
3. 在 `agent_factory.py` 中创建智能体
4. 在 `workflow.py` 中添加工作流连接

### 集成新的MCP工具

1. 在 `mcp.json` 中添加工具配置
2. 确保工具依赖已安装
3. 在相应的智能体提示词中添加工具使用说明

## 故障排除

### 常见问题

1. **MCP服务器启动失败** - 检查工具依赖和环境配置
2. **API调用失败** - 验证API密钥和网络连接
3. **报告生成失败** - 检查输出目录权限

### 日志调试

系统会输出详细的执行日志，包括：
- 智能体创建状态
- MCP服务器启动情况
- 分析执行进度
- 报告生成结果

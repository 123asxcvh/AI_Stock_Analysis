# 股票网站 - 股票数据管理与分析系统

一个综合性的股票数据管理与分析系统，支持A股和H股的数据采集、分析和展示。

## 🎯 项目目标

本项目旨在提供一个完整的股票数据分析解决方案，支持：
- **数据采集**：A股和H股数据采集
- **数据分析**：股票数据分析和处理
- **数据存储**：数据存储和管理
- **数据展示**：数据可视化和展示
- **策略分析**：投资策略分析


## 🚀 快速开始

```bash
# A股分析
cd a股
python main.py

# H股分析
cd h股
python src/main.py
```

## 🔧 核心功能

### A股功能
- **数据采集**：A股市场数据采集
- **技术分析**：技术指标分析
- **基本面分析**：基本面数据分析
- **策略回测**：投资策略回测

### H股功能
- **数据管理**：H股数据管理
- **实时监控**：实时数据监控

## 📊 支持的市场

| 市场 | 代码格式 | 示例 | 支持状态 |

## 🔧 技术栈

### 核心技术
- **Python 3.8+**：主要编程语言
- **pandas**：数据处理和分析
- **numpy**：数值计算
- **matplotlib**：图表绘制
- **requests**：数据获取

### 数据源
- **Tushare**：A股数据源
- **AkShare**：多市场数据源
- **Yahoo Finance**：H股数据源
